# 小林畳商店 公式ウェブサイト

大阪府大阪市福島区の畳専門店「小林畳商店」公式ウェブサイトのソースコードです。

## 技術スタック

- **フロントエンド**: Astro
- **ホスティング**: Cloudflare Pages + Cloudflare Workers
- **バージョン管理**: Git + GitHub
- **デプロイ**: Git連携による自動デプロイ

## 推奨ブラウザ

- Google Chrome メジャーバージョン 144以上
- Firefox メジャーバージョン 147以上
